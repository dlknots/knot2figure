# knot2figure
系北京师范大学北创项目——“基于纽结等价的深度学习方法”项目组成果
## introduction
- knot2figure2knot.py
——实现纽结图或链环图与三维参数曲线间的相互转换
- example.py
——纽结转化示例
## functions
- fun1()——实现纽结图向三维参数曲线的转化
- fun2()——同时实现纽结与链环的转化问题
- knot2figure()——实现三维参数曲线向纽结图or链环图的转化
## problems
- 代码暂时不能解决圆环，即最简单的平凡纽结的三维曲线转化问题
- 代码运行速度较慢，平均用时10秒左右
